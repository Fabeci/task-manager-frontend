Configuration du Frontend (Vue.js)
Naviguez vers le répertoire du frontend :

cd task-manager-frontend
Installez les dépendances :

npm install
Si vous utilisez Yarn :

yarn install
Démarrez le serveur de développement :

npm run serve
Si vous utilisez Yarn :

yarn serve
Votre application frontend sera disponible à http://localhost:8080 par défaut.

Utilisation
Inscription et connexion : Accédez aux pages d'inscription et de connexion via le menu de navigation.
Gestion des tâches : Une fois connecté, vous pourrez créer, modifier et supprimer des tâches.
Technologies utilisées
Backend : Laravel 10
Frontend : Vue.js 5.0.8 avec TailwindCSS
